# Background Music Tracks

Curated royalty-free tracks for App Store preview videos. The appshot skill auto-selects based on app category.

## Track List

Download each track (30 seconds, MP3), trim to 30s if longer, and save with the filename below.

## Requirements

- Format: MP3, 128-192kbps
- Duration: 25-30 seconds (trim longer tracks)
- No vocals, no sudden beat drops
- Soft enough to work at volume 0.2-0.3 without competing with visual storytelling

## Where to find tracks

1. **Pixabay Music** (https://pixabay.com/music/) — Free, commercial, no attribution
2. **Chosic CC0** (https://www.chosic.com/free-music/all/?attribution=no) — Public domain
3. **Mixkit** (https://mixkit.co/free-stock-music/) — Free, commercial, no attribution

Use Pixabay as the primary source. All tracks there are free for commercial use including App Store videos.
